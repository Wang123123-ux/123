# 🥗 减脂饮食管理软件

## 🚀 快速启动

**双击运行 `启动软件.bat` 即可打开软件！**

无需任何配置，打开即用。

---

## 界面功能

### 📅 今日概览
- 查看任意日期的饮食记录
- 实时显示热量和营养素总计
- 热量摄入进度条

### 📝 添加食物
1. 选择餐次（早餐/午餐/晚餐/加餐）
2. 选择食物分类
3. 选择具体食物
4. 输入重量（克）
5. 点击"添加记录"

### 📋 记录管理
- 查看当日所有饮食记录
- 可删除误添加的记录

### 💪 营养分析
- 自动计算三大营养素占比
- 帮助调整饮食结构

## 故障排除

1. 确认已安装 Python 3.7+
2. 右键 `启动软件.bat` → 以管理员身份运行
3. 检查杀毒软件是否拦截
