# 🥗 减脂期饮食管理软件

一个简单易用的减脂期饮食管理工具，帮助你记录每日饮食、计算热量摄入和追踪三大宏量营养素。

## 功能特点

- 📝 **饮食记录** - 记录每日三餐和加餐
- 🔥 **热量计算** - 自动计算每日总热量摄入
- 💪 **宏量营养分析** - 追踪蛋白质、脂肪、碳水化合物摄入
- 🍎 **食物数据库** - 内置 80+ 种常见食物营养数据
- 📊 **数据持久化** - 自动保存饮食记录到本地

## 安装与使用

### 环境要求
- Python 3.7+

### 使用方法

1. 进入项目目录：
```bash
cd diet-tracker
```

2. 运行程序：
```bash
python main.py
```

### 功能菜单

```
  1. 📝 记录饮食
  2. 📊 查看今日摄入
  3. 📅 查看指定日期记录
  4. 🔍 查询食物营养
  5. 📈 查看食物分类
  6. ⚙️  设置每日目标
  7. 💾 保存并退出
```

## 数据结构

### 饮食记录格式

每个食物记录包含：
- 食物名称
- 重量（克）
- 餐次类型（早餐/午餐/晚餐/加餐）
- 营养成分（热量、蛋白质、脂肪、碳水）

### 数据存储

数据保存在 `diet_data.json` 文件中，采用 JSON 格式。

## 减脂期营养建议

| 营养素 | 建议摄入 | 说明 |
|--------|----------|------|
| 热量 | 1200-1600 kcal | 根据基础代谢调整 |
| 蛋白质 | 1.2-1.6g/kg | 保持肌肉量 |
| 脂肪 | 40-60g | 优先不饱和脂肪 |
| 碳水 | 100-150g | 选择复合碳水 |

## 扩展数据库

如需添加新食物，编辑 `food_database.py` 中的 `FOOD_DATABASE` 字典：

```python
"食物名称": {"calories": 热量，"protein": 蛋白质，"fat": 脂肪，"carbs": 碳水},
```
